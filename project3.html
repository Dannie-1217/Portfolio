ENVIROSAVER

Interactive Educational Toy

Video Description

The demo video of the EnviroSaver

EnviroSaver is a sensor-driven educational toy designed to teach children about environmental conservation through interactive play. Completed by a four-person team for IAT 267, I developed all technical components — including Arduino sensor logic, motor control, LED feedback, and the Processing interface that synchronizes animations with physical interactions.

01

Concept Development

The system was designed to teach environmental habits through playful interaction. We designed three modules—waste disposal, water saving, and electricity conservation—each powered by real-time sensor input and physical feedback. My initial concept guided the project direction, framing the interaction goals and technical requirements.

02

Design & Prototyping

We built the system using Arduino, LEDs, motors, and multiple sensors including IR, rotation, and light sensors. I implemented the entire technical workflow — wiring, sensor calibration, testing, and core interactive logic. Each sensor triggered specific animations and audio cues within the Processing interface.

03

Technical Development

I programmed Arduino to read multi-sensor inputs and control motors and LED feedback. I also developed the Processing interface, enabling real-time communication between hardware and software. This ensured that every physical action — dropping waste, turning off a light, saving water — immediately triggered a synchronized digital response. Additionally, I implemented sound effects that react to the player's physical interactions, enhancing engagement and reinforcing feedback through audio cues.

04

Final Product

The final system delivered an engaging multisensory experience. Children interacted with physical components while receiving visual, audio, and LED feedback, successfully blending physical play with interactive digital elements. This project strengthened my abilities in hardware and software integration, system logic design, and rapid prototyping.
