STYLE SKETCH

Interactive Painting with Intelligent Generation

Video Description

A demonstration of the Garage e-commerce website functionality including browsing, cart interaction, and checkout.

A project for IAT 461: Generative AI and Computational Creativity — an interactive sketch-to-image system that lets users draw, write prompts, and instantly generate stylized AI paintings.

01

Introduction

StyleSketch is an interactive AI drawing system that combines Sketch + Prompt to generate fully stylized images using Stable Diffusion, ControlNet-like structure guidance, and LoRA style modules. Users can draw a sketch, write a prompt, enhance it with an optional AI model (Mistral-7B), and generate results in different art styles instantly.

02

Sketch-to-Image Pipeline

Users draw a sketch on the canvas or upload an image. StyleSketch processes the sketch and feeds it into a Stable Diffusion img2img pipeline with structure-aware guidance. This allows users to control both the composition and the creative direction of the final image.

03

Prompt Enhancement(Mistral-7B)

To support beginners, StyleSketch includes an optional prompt enhancer powered by a locally quantized Mistral-7B (4-bit) model. With a single toggle, the system rewrites the user's prompt into a richer, more descriptive version— helping produce more detailed and consistent images.

04

LoRA Style Switching

The system supports dynamic LoRA loading, allowing users to switch between multiple art styles such as Van Gogh, Cyberpunk, and Clay Animation. Each style is applied instantly without restarting the model pipeline, creating a playful real-time exploration experience.

05

Interactive UI & Workflow Design

The interface was designed to encourage a co-creative workflow. Users can draw, write prompts, adjust parameters, preview results, and iterate quickly. This focus on interaction transforms the model from a static generator into an intuitive visual playground.
